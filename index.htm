Learning Postgres DB:

PostgreSQL is a powerful, open source object-relational database system. First thing we need to do is to start the postgres service by using the following ( sudo service postgresql start ) command in Linux terminal in Cloud 9 as showing in the photo below.

​ 

After running the command, we can now use the following code  sudo sudo -u postgres psql Connect to the service.

Since this is the first time to setup the postgres so we need to setup a password for our root user which is postgres, and to do that we can use the following command ( \password ) the system should ask you to enter a new password and after to do that it should ask you to re-enter the password as showing in the following photo:

Now, if we disconnect our connection from the postgres service using this command (\q), we can simply reconnecting the connection using the following command                        ( psql -U postgres -h localhost ) then the system should ask you to enter the password you created before.

Now let start to learn how to create database and tables using psql:

To create a database in psql all we have to do is using this sql command

CREATE DATABASE database_name;

, sql is case insinsteve, however it is preferable to use capital letters for all sql key words.

Now that we have a database, we can now create tables inside our database but before that we must use the this command (\c database_name) to connect to our database.

After connecting to our database, we can simply start creating our table/s using the folloiwng  sql command

CREATE TABLE table_name (
    code        int PRIMARY KEY,
    title       varchar(40) NOT NULL,
    id        integer NOT NULL,
    
);

( CREATE TABLE table_name (id int, name text); )
